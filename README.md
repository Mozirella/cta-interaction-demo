# CTA Interaction Demo

This project demonstrates basic frontend interaction using HTML, CSS, and JavaScript.

Features:
- Styled CTA button
- Hover effects
- Click interaction with dynamic message display

This reflects real-world UI adjustments and interaction handling similar to work done in production environments.
